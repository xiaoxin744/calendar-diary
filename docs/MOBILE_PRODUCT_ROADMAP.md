# 移动端产品与技术路线

## 第一阶段：可发布 MVP

- 月历、日记、心情、月度计划与全文搜索
- 本地优先存储、备份恢复、生物识别锁
- Android/iOS/Web 构建与持续集成
- 崩溃边界、数据校验、无障碍标签和隐私说明

## 第二阶段：跨设备能力

- ✅ 与桌面端共用 WebDAV 同步协议
- ✅ 同步前自动快照、三方冲突对比和逐项合并
- ✅ 系统后台调度、前台定时同步和并发写入重试
- 待办：完整离线操作队列和同步历史审计页面
- 系统日历只读/双向集成与提醒通知

## 第三阶段：组织级质量

- Sentry 等可自托管崩溃监控，默认不采集日记正文
- 端到端加密、系统凭据库和密钥轮换
- Maestro/Detox 端到端测试与真实设备矩阵
- 分阶段发布、远程配置、回滚和版本兼容策略
- WCAG 对比度、动态字体、屏幕阅读器和多语言

## 功能优先级原则

1. 不丢数据高于新增功能。
2. 默认本地与最小权限高于增长追踪。
3. 跨端数据协议稳定高于界面快速迭代。
4. 每项云功能必须提供关闭开关和明确的数据边界。
