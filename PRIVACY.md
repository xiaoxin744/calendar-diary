# 隐私说明

生效日期：2026-07-15

日历日记采用本地优先设计，不接入广告、用户画像或第三方行为统计服务。

## 数据处理

- 日记、待办、心情和月度计划默认只保存在用户当前设备。
- 导出备份时，应用仅按用户操作生成 JSON 文件并交给系统保存或分享。
- 桌面端或移动端启用 WebDAV 后，数据会发送到用户自行配置的 WebDAV 服务；该服务的隐私与安全规则由其提供方决定。
- 移动端 WebDAV 密码在 Android/iOS 上保存在系统安全凭据库；Web 版保存在当前浏览器的本地存储中。后台同步只访问用户配置的 WebDAV 和应用本地数据。
- 设备解锁功能调用 Android 或 iOS 的系统认证界面。应用只接收认证成功或失败结果，不读取、保存指纹或面容数据。
- 当前版本不采集崩溃日志、设备标识符、位置、通讯录、广告标识符或日记正文。

## 数据删除

用户可以在应用设置中清空本地数据，也可以通过操作系统卸载应用并删除其本地数据。WebDAV 上的远程备份需要由用户在同步界面或对应服务中删除。

## 权限

应用仅在用户主动启用相关功能时请求文件访问、系统分享或设备认证能力。权限可以在操作系统设置中撤销。

## 变更与联系

本说明发生实质变更时，会在仓库更新日志中记录。隐私或安全问题请通过 GitHub 仓库的 Issue 或私密漏洞报告入口联系维护者，且不要附上日记正文、密码、令牌或备份文件。
